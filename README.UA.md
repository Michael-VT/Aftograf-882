# Відладчик-симулятор Автограф-882

Інтерактивний браузерний відладчик і симулятор для графобудівника **Автограф-882** на базі мікропроцесора **К580ІК80А** (клон Intel 8080).

Проєкт являє собою повний цифровий двійник оригінального пристрою: емуляцію CPU, пам'ять з відображеним введенням-виведенням, дизасемблер, симуляцію плоттера, USART-термінал і завантажувач HPGL — усе працює в браузері без серверної логіки.

## Можливості

### Емуляція CPU (cpu8080.js)
- Повна емуляція К580ІК80А / Intel 8080 — усі 256 опкодів, табличний метод
- Регістри: A, B, C, D, E, H, L, SP, PC
- Прапорці: S, Z, AC, P, CY (бітові позиції 8080)
- Обробка переривань (INTR з вектором RST)
- Підрахунок T-станів
- Регулятор швидкості: від max (безліміт) до 100 Гц

### Пам'ять (memory.js)
- ПЗП: 24 КБ за адресою `$0000–$5FFF` (три мікросхеми D2764A)
- ОЗП: 1 КБ за адресою `$6000–$63FF` (КР537РУ10)
- I/O з відображенням на пам'ять: PPI1 за `$E000`, PPI2 за `$E400`, PIT за `$E800`, USART за `$EC00`
- Читання з невідображених адрес повертає `$FF`; запис у ПЗП або невідображену область логується

### Дизасемблер
- Гібридний рекурсивно-лінійний дизасемблер на основі таблиці опкодів CPU
- 6 колонок: точка зупину, адреса, байти, мнемоніка, операнди, анотація
- Follow-PC: підсвічування поточної інструкції
- Віртуальне прокручування всіх 64 КБ пам'яті
- Клік для встановлення breakpoint, подвійний клік для переходу PC
- Пошук за адресою (клавіша `J` — стрибок до рядка під курсором)

### Перегляд пам'яті
- Віртуальне прокручування всіх 64 КБ адресного простору
- Кольорове маркування: ПЗП (сірий), ОЗП (жовтий), I/O (фіолетовий)
- Редагування байтів по кліку (Tab для наступного байта)
- Підсвічування покажчика HL помаранчевим маркером
- Рядок адреси для швидкої навігації

### Симуляція плоттера
- Симуляція крокових двигунів XY з фаз порту PPI
- 7 кольорів пера (з аналізу прошивки)
- Полотно A4 портрет (1:√2) з підтримкою Retina
- Сітка з автоматичним масштабуванням, курсор поточної позиції
- Кнопки очищення полотна та автомасштабу

### Завантажувач HPGL
- Завантаження HPGL-файлів: команди `IN`, `SP`, `PU`, `PD`
- **Прямий рендер**: розбір і відмальовування з анімацією
- **Режим UART**: посимвольне відправлення HPGL в USART для обробки прошивкою
- Індикатор прогресу та пауза/продовження

### USART-термінал
- Поле введення hex-байтів для відправки CPU
- Завантаження файлу з передачею по XOn-XOff
- Лог передачі з відображенням друкованих символів та hex-кодів
- Індикація TXRDY/RXRDY

### Сесії
- Повний знімок стану CPU, ОЗП, точок зупину та ліній плоттера
- Збереження в JSON з міткою часу
- Відновлення з раніше збереженої сесії

### Довідка
- Кнопка `?` та клавіші `?`/`/` відкривають вікно довідки
- Таблиця клавіатурних скорочень
- Довідник дій миші
- Огляд форматів файлів

### Теми оформлення
- Темна тема (за замовчуванням) — палітра Tokyo Night
- Світла тема — чиста світла палітра для денної роботи
- Перемикання в панелі налаштувань, зберігається в `localStorage`

## Запуск

```bash
cd ~/work/Antigravity/github/aftograf
python3 -m http.server 8080
```

Відкрий `http://localhost:8080/sim/` у браузері.

Прошивка (`firmware.bin`, 24 КБ) завантажується автоматично.  
Якщо відсутня — використовуй кнопку 📂 або Налаштування → Завантажити прошивку.

## Клавіатурні скорочення

| Клавіша | Дія |
|---|---|
| `Пробіл` / `→` | Крок (одна інструкція) |
| `R` | Скидання CPU |
| `F5` | Пуск / Пауза |
| `B` | Breakpoint на PC |
| `J` | Перехід PC до адреси під курсором |
| `?` / `/` | Відкрити довідку |
