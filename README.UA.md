# Відладчик-симулятор Автограф-882 v1.0.11

*Оригінальний графопобудовник Автограф-882*

## Можливості

### Емуляція CPU
- Повна емуляція К580ІК80А / Intel 8080 — всі 256 опкодів
- Регістри: A, B, C, D, E, H, L, SP, PC (редагуються)
- Прапорці: S, Z, AC, P, CY
- Обробка переривань (INTR з вектором RST)
- Лічильник тактів

### Пам'ять
- ПЗП: 24 КБ за адресою `$0000–$5FFF` (D2764A)
- ОЗП: 2 КБ за адресою `$6000–$67FF` (КР537РУ10)
- I/O: PPI1 за `$E000`, PPI2 за `$E400`, PIT за `$E800`, USART за `$EC00`

### Дизасемблер
- 256 інструкцій на екрані, доступ до всіх 64 КБ
- **Follow PC** — поточна інструкція завжди по центру
- Пошук за адресою, кнопки ◀▶
- Клік для breakpoint

### Перегляд пам'яті
- 64 рядки × 16 байт = 1 КБ видимо
- Навігація: адресний рядок + Go, ◀▶, HL
- Редагування байтів подвійним кліком
- ASCII колонка праворуч

### HPGL
- Команди: IN, SP, PU, PD, PA, PR
- **Preview**: малювання всього файлу
- **Покроковий режим**: ▶ Next / ▶▶ All / ⟲ Reset
- **Намалювати до N**: введення номера сегмента
- Прогрес-бар, підсвітка активного рядка

## Складання та запуск

```bash
cd rust
cargo run --release
```

### Тести

```bash
cd rust
cargo test -- --test-threads=1
```

### Браузерна версія

```bash
python3 -m http.server 8080
# Відкрий http://localhost:8080/sim/
```

## Структура проекту

```
├── rust/                  ← Основна версія (Rust)
│   ├── Cargo.toml
│   ├── build.rs
│   ├── TESTS.md
│   └── src/
│       ├── main.rs
│       ├── app.rs         ← UI, кроки, I/O
│       ├── cpu.rs         ← Емулятор 8080
│       ├── memory.rs      ← MMU
│       ├── disasm.rs      ← Дизасемблер
│       ├── plotter.rs     ← Плоттер
│       ├── hpgl.rs        ← Парсер HPGL
│       ├── ppi8255.rs     ← КР580ВВ55А
│       ├── pit8253.rs     ← КР580ВІ53
│       ├── usart8251.rs   ← КР580ВВ51А
│       ├── settings.rs    ← Налаштування
│       └── session.rs     ← Збереження сесії
├── sim/                   ← Браузерна версія
├── docs/                  ← Документація
└── images/                ← Скріншоти
```

## Клавіші

| Клавіша | Дія |
|---------|-----|
| `Пробіл` / `→` | Крок |
| `R` | Скидання CPU |
| `F5` | Пуск / Пауза |
| `B` | Breakpoint |
| `J` | Перехід PC |
