# Отладчик-симулятор Автограф-882

Интерактивный браузерный отладчик и симулятор для графопостроителя **Автограф-882** на базе микропроцессора **К580ИК80А** (клон Intel 8080).

Проект представляет собой полный цифровой двойник оригинального устройства: эмуляцию CPU, память с отображённым вводом-выводом, дизассемблер, симуляцию плоттера, USART-терминал и загрузчик HPGL — всё работает в браузере без серверной логики.

## Возможности

### Эмуляция CPU (cpu8080.js)
- Полная эмуляция К580ИК80А / Intel 8080 — все 256 опкодов, табличный метод
- Регистры: A, B, C, D, E, H, L, SP, PC
- Флаги: S, Z, AC, P, CY (битовые позиции 8080)
- Обработка прерываний (INTR с вектором RST)
- Подсчёт T-состояний
- Регулятор скорости: от max (безлимит) до 100 Гц

### Память (memory.js)
- ПЗУ: 24 КБ по адресу `$0000–$5FFF` (три микросхемы D2764A)
- ОЗУ: 1 КБ по адресу `$6000–$63FF` (КР537РУ10)
- I/O с отображением на память: PPI1 по `$E000`, PPI2 по `$E400`, PIT по `$E800`, USART по `$EC00`
- Чтение из неотображённых адресов возвращает `$FF`; запись в ПЗУ или неотображённую область логируется

### Дизассемблер
- Гибридный рекурсивно-линейный дизассемблер на основе таблицы опкодов CPU
- 6 колонок: точка останова, адрес, байты, мнемоника, операнды, аннотация
- Follow-PC: подсветка текущей инструкции
- Виртуальная прокрутка всей 64 КБ памяти
- Клик для установки breakpoint, двойной клик для перехода PC
- Поиск по адресу (клавиша `J` — прыжок к строке под курсором)
- Копирование видимого диапазона в буфер обмена

### Просмотр памяти
- Виртуальная прокрутка всех 64 КБ адресного пространства
- Цветовая маркировка: ПЗУ (серый), ОЗУ (жёлтый), I/O (фиолетовый)
- Редактирование байтов по клику (Tab для следующего байта)
- Подсветка указателя HL оранжевым маркером
- Строка адреса для быстрой навигации

### Эмуляция периферии
- **КР580ВВ55А (PPI8255)**: две микросхемы, 3 порта + регистр управления
- **КР580ВИ53 (PIT8253)**: 3 × 16-битных счётчика с фиксацией
- **КР580ВВ51А (USART8251)**: буферы RX/TX с управлением XOn-XOff, генерация прерываний

### Симуляция плоттера
- Симуляция шаговых двигателей XY из фаз порта PPI
- 7 цветов пера (из анализа прошивки)
- Холст в формате A4 (1:√2) с поддержкой Retina
- Сетка с автоматическим масштабированием, курсор текущей позиции
- Кнопки очистки холста и автомасштаба

### Загрузчик HPGL
- Загрузка HPGL-файлов: команды `IN`, `SP`, `PU`, `PD`
- **Прямой рендер**: разбор и отрисовка с анимацией
- **Режим UART**: посимвольная отправка HPGL в USART для обработки прошивкой
- Индикатор прогресса и пауза/продолжение

### USART-терминал
- Поле ввода hex-байтов для отправки CPU (например, `01 02 FF`)
- Загрузка файла с передачей по XOn-XOff
- Лог передачи с отображением печатных символов и hex-кодов
- Индикация TXRDY/RXRDY

### Сохранение и загрузка сессий
- Полный снимок состояния CPU, ОЗУ, точек останова и линий плоттера
- Сохранение в JSON с меткой времени
- Восстановление из ранее сохранённой сессии

### Подсказка
- Кнопка `?` в заголовке и клавиши `?`/`/` открывают окно подсказки
- Таблица клавиатурных сокращений
- Справочник действий мыши
- Обзор форматов файлов
- Быстрые советы

### Темы оформления
- Тёмная тема (по умолчанию) — палитра Tokyo Night
- Светлая тема — чистая светлая палитра для дневной работы
- Переключение в панели настроек, сохраняется в `localStorage`

### Панель настроек
- Настройка адресов отслеживаемых переменных (X, Y, перо, цвет)
- Пользовательские переменные наблюдения (1 или 2 байта)
- Конфигурация смещений чипов ROM
- Загрузка прошивки с выбором адреса
- Выбор темы
- Все настройки сохраняются в `localStorage`

## Запуск

```bash
cd ~/work/Antigravity/github/aftograf
python3 -m http.server 8080
```

Открой `http://localhost:8080/sim/` в браузере.

Прошивка (`firmware.bin`, 24 КБ) загружается автоматически при старте.  
Если отсутствует — используй кнопку 📂 или Настройки → Загрузить прошивку.

## Сборка bundle.js

```bash
cd sim && node build.js
```

Собирает единый `bundle.js` из ES6-модулей (`settings.js`, `memory.js`, `cpu8080.js`, `main.js`) — удаляет `import`/`export` и конкатенирует в порядке зависимостей.

## Клавиатурные сокращения

| Клавиша | Действие |
|---|---|
| `Пробел` / `→` | Шаг (одна инструкция) |
| `R` | Сброс CPU |
| `F5` | Пуск / Пауза |
| `B` | Breakpoint на PC |
| `J` | Переход PC к адресу под курсором |
| `?` / `/` | Открыть подсказку |

## Структура проекта

```
├── sim/                  ← Браузерный отладчик
│   ├── index.html        Точка входа (3-колоночный макет)
│   ├── styles.css        Система тёмной/светлой темы
│   ├── bundle.js         Сборка (единый файл)
│   ├── build.js          Скрипт сборки
│   ├── main.js           Контроллер приложения и stubs периферии
│   ├── cpu8080.js        Эмулятор CPU К580ИК80А
│   ├── memory.js         MMU с ПЗУ, ОЗУ, I/O
│   ├── settings.js       Менеджер настроек + localStorage
│   └── firmware.bin      Образ прошивки 24 КБ
├── disasm8080.py         Дизассемблер на Python
├── autograf-882-disassembly.asm  Полный листинг дизассемблера
├── 01_Plotter-*-Schematic.pdf    Реверсированная принципиальная схема
└── CHECKPOINT.md         Чекпоинт разработки
```
