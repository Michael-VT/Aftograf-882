# Отладчик-симулятор Автограф-882 v1.0.15

![Автограф-882 — Оригинальное устройство](images/%D0%90%D0%B2%D1%82%D0%BE%D0%B3%D1%80%D0%B0%D1%84_882.01-1990.jpg)
*Оригинальный графопостроитель Автограф-882*

## Возможности

### Эмуляция CPU (Rust & Go)
- Полная эмуляция К580ИК80А / Intel 8080 — все 256 опкодов
- Регистры: A, B, C, D, E, H, L, SP, PC (редактируемые hex-поля)
- Флаги: S, Z, AC, P, CY (кликабельные)
- Обработка прерываний (INTR с вектором RST 7)
- Счётчик тактов в панели CPU
- Множитель скорости: 1x/10x/100x/1Kx/10Kx/100Kx (Go)

### Память
- ПЗУ: 24 КБ по адресу `$0000–$5FFF` (D2764A)
- ОЗУ: 2 КБ по адресу `$6000–$67FF` (КР537РУ10)
- I/O: PPI1 по `$E000`, PPI2 по `$E400`, PIT по `$E800`, USART по `$EC00`

### Дизассемблер (Rust & Go)
- Табличный дизассемблер на все 256 опкодов
- Breakpoint (клик для установки/снятия)
- **Follow PC** — текущая инструкция по центру
- Поиск по адресу, кнопки ◀▶
- Клик по адресу — прыжок в просмотрщик памяти
- Копирование видимого диапазона в буфер (Go: кнопка Copy)

### Просмотр памяти (Rust & Go)
- 32 строки × 16 байт = 512 Б видимо (Go); 64 строки (Rust)
- Навигация: строка адреса + Go, ◀▶, HL-кнопка
- Клик по BC:/DE:/HL:/SP: — прыжок к адресу
- Цветовая маркировка: ПЗУ (коричневый), ОЗУ (золотой), I/O (фиолетовый) — Go
- Редактирование байтов кликом (Enter для подтверждения)
- ASCII колонка справа

### Периферия (Rust & Go)
- **КР580ВВ55А (PPI8255)**: две микросхемы (PPI1, PPI2)
- **КР580ВИ53 (PIT8253)**: 3 счётчика
- **КР580ВВ51А (USART8251)**: буферы, отправка hex, лог

### Плоттер (Rust & Go)
- Шаговые двигатели XY из фаз PPI
- 7 цветов пера
- Холст A4, автошкала
- Загрузка HPGL, пошаговое исполнение

### HPGL
- Команды: IN, SP, PU, PD, PA, PR
- **Preview**: рисование всего файла
- **Шаговый режим**: ▶ Next / ▶▶ All / ⟲ Reset
- Прогресс-бар

### USART (Go)
- Поле hex-ввода, отправка байтов в CPU
- Лог приёма (последние 20 записей)
- Индикаторы TXRDY/RXRDY

### Диагностика
- Панель CPU со счётчиком тактов
- Стек (8 слов в Go)
- DIP-светодиоды (PPI1 port A)
- Сохранение/загрузка сессии в JSON (Go)
- Горячие клавиши: Space/→ шаг, R сброс, F5 пуск/пауза, B breakpoint, ? помощь

## Сборка и запуск

### Rust (основная версия)

```bash
cd rust
cargo run --release
```

Тесты:

```bash
cd rust
cargo test -- --test-threads=1
```

### Go (активно развивается)

```bash
cd go
go run ./cmd/aftograf
```

Версия Go использует Fyne v2.5 для GUI. Требуется дисплейный сервер (X11/macOS/Wayland).

Тесты:

```bash
cd go
go test ./...
```

### Браузерная версия

`sim/` — старая версия для браузера:

```bash
python3 -m http.server 8080
# Открой http://localhost:8080/sim/
```

│   │   ├── disasm/        Дизассемблер + тесты
│   │   ├── plotter/       Плоттер
│   │   ├── hpgl/          Парсер HPGL
│   │   ├── ppi8255/       PPI8255
│   │   ├── pit8253/       PIT8253
│   │   ├── usart8251/     USART8251
│   │   └── settings/      Настройки
│   └── assets/firmware.bin

## Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| `Пробел` / `→` | Шаг |
| `R` | Сброс CPU |
| `F5` | Пуск / Пауза |
| `B` | Breakpoint |
| `?` | Помощь |
