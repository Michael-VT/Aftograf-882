# Отладчик-симулятор Автограф-882 v1.0.10

![Автограф-882 — Оригинальное устройство](images/%D0%90%D0%B2%D1%82%D0%BE%D0%B3%D1%80%D0%B0%D1%84_882.01-1990.jpg)
*Оригинальный графопостроитель Автограф-882*

## Возможности

### Эмуляция CPU
- Полная эмуляция К580ИК80А / Intel 8080 — все 256 опкодов
- Регистры: A, B, C, D, E, H, L, SP, PC (редактируемые)
- Флаги: S, Z, AC, P, CY
- Обработка прерываний (INTR с вектором RST)
- Счётчик тактов в панели CPU

### Память
- ПЗУ: 24 КБ по адресу `$0000–$5FFF` (D2764A)
- ОЗУ: 2 КБ по адресу `$6000–$67FF` (КР537РУ10)
- I/O: PPI1 по `$E000`, PPI2 по `$E400`, PIT по `$E800`, USART по `$EC00`

### Дизассемблер
- 256 инструкций на экране, полный доступ к 64 КБ
- **Follow PC** — текущая инструкция всегда по центру
- Поиск по адресу, кнопки ◀▶ для навигации
- Клик для breakpoint, двойной клик для перехода

### Просмотр памяти
- 64 строки × 16 байт = 1 КБ видимо
- Навигация: строка адреса + Go, ◀▶, HL-кнопка
- Клик по регистрам BC:/DE:/HL:/SP: — прыжок к адресу
- Цветовая маркировка регионов
- Редактирование байтов двойным кликом
- ASCII колонка справа

### Периферия
- **КР580ВВ55А (PPI8255)**: две микросхемы
- **КР580ВИ53 (PIT8253)**: 3 счётчика
- **КР580ВВ51А (USART8251)**: буферы с XOn-XOff

### Плоттер
- Шаговые двигатели XY из фаз PPI
- 7 цветов пера
- Холст A4 (1:√2), автошкала, сетка

### HPGL
- Команды: IN, SP, PU, PD, PA, PR
- **Preview**: рисование всего файла
- **Шаговый режим**: ▶ Next / ▶▶ All / ⟲ Reset
- **Нарисовать до N**: ввод номера сегмента
- Прогресс-бар, подсветка активной строки

### Сенсоры и датчики
- **DIP-переключатели (D4-D7)**: в панели Sensors
- **Концевые датчики (D3-D0)**: переключаемые
- **LED индикаторы**: биты порта C PPI1

## Сборка и запуск

```bash
cd rust
cargo run --release
```

### Тесты

```bash
cd rust
cargo test -- --test-threads=1
```

### Браузерная версия

`sim/` — старая версия для браузера:

```bash
python3 -m http.server 8080
# Открой http://localhost:8080/sim/
```

## Структура проекта

```
├── rust/                  ← Основная версия (Rust)
│   ├── Cargo.toml
│   ├── build.rs
│   ├── TESTS.md
│   └── src/
│       ├── main.rs
│       ├── app.rs         ← UI, шаги, I/O
│       ├── cpu.rs         ← Эмулятор 8080
│       ├── memory.rs      ← MMU
│       ├── disasm.rs      ← Дизассемблер
│       ├── plotter.rs     ← Плоттер
│       ├── hpgl.rs        ← Парсер HPGL
│       ├── ppi8255.rs     ← КР580ВВ55А
│       ├── pit8253.rs     ← КР580ВИ53
│       ├── usart8251.rs   ← КР580ВВ51А
│       ├── settings.rs    ← Настройки
│       └── session.rs     ← Сохранение сессии
├── sim/                   ← Браузерная версия
├── docs/                  ← Документация
├── images/                ← Скриншоты
└── *.hpgl                 ← Примеры HPGL
```

## Клавиши

| Клавиша | Действие |
|---------|----------|
| `Пробел` / `→` | Шаг |
| `R` | Сброс CPU |
| `F5` | Пуск / Пауза |
| `B` | Breakpoint |
| `J` | Переход PC |
| `?` / `/` | Подсказка |
